# 📌 О программе

**Jarvis Assistant** — это голосовой ассистент для автоматизации рутинных задач на ПК.

Программа слушает команды через микрофон, распознаёт речь на русском языке и выполняет действия: открывает приложения, управляет YouTube, ведёт заметки, выполняет поиск в Google и управляет окнами.

Ассистент автоматически:

* слушает микрофон и распознаёт голосовые команды
* произносит ответы через динамики (синтез речи)
* открывает программы и сайты по команде
* управляет YouTube-окнами (пауза, полный экран)
* ведёт список заметок (добавление, просмотр, удаление)
* выполняет поиск в Google
* переходит в режим сна и просыпается по команде

---

# ⚙️ Основные возможности

## Распознавание речи

Ассистент слушает микрофон в реальном времени и распознаёт русскоязычные команды через Google Speech Recognition.

---

## Синтез речи

Ответы произносятся вслух через Google Text-to-Speech (gTTS) прямо через динамики.

---

## Управление приложениями

По голосовой команде ассистент открывает:

* YouTube (в браузере)
* Telegram
* Obsidian
* Steam
* Chrome
* Рабочие программы (папка WORK)

---

## Управление YouTube

Ассистент умеет:

* открывать каналы конкретных авторов
* искать видео по запросу
* ставить на паузу / снимать с паузы
* переключать полноэкранный режим

---

## Заметки

Программа ведёт текстовый список дел в файле `notes.txt`:

* просмотр всех заметок
* добавление новой заметки
* удаление заметки по голосу

---

## Поиск в Google

По команде открывает браузер с результатами поиска по произнесённому запросу.

---

## Режим сна

Ассистент поддерживает два состояния:

* **Активный режим** — полноценно обрабатывает все команды
* **Режим сна** — ждёт команды пробуждения, игнорируя остальные

---

## Режим повторения

После активации ассистент повторяет за пользователем всё произнесённое вслух.

---

## Управление окнами

Специальный режим для управления открытыми окнами: пауза/воспроизведение и сворачивание/разворачивание YouTube.

# 🎙️ Голосовые команды

## Приветствие

```
привет
привет джарвис
джарвис привет
```

---

## Открыть YouTube

```
включи youtube
запусти ютубчик
давай посмотрим youtube
```

После команды ассистент спросит, что хочешь посмотреть.

---

## Открыть программы для работы

```
программы для работы
запусти проги для работы
```

---

## Заметки

```
# Просмотреть
покажи заметки
какие есть заметки
напомни мои дела

# Добавить
добавь заметку
сделай заметку
новая заметка

# Удалить
удали заметку
убери заметку
вычеркнуть дело
```

---

## Поиск в Google

```
поиск
есть вопрос
джарвис ищи
найди для меня кое-что
```

После команды произнеси запрос — откроется браузер с результатами.

---

## Режим управления окнами

```
управление окнами
джарвис управление окнами
режим управления окнами
```

В этом режиме доступны команды для YouTube:

```
# Пауза / Воспроизведение
пауза
стоп
продолжай

# Полный экран / Свернуть
разверни
сверни
полноэкранный
```

---

## Режим сна

```
# Уснуть
джарвис засыпай
джарвис уходи в покой
засыпай
джарвис режим сна

# Проснуться
джарвис просыпайся
джарвис вставай
просыпаемся
джарвис активный режим
```

---

## Режим повторения

```
# Включить
повтори за мной
повторяй за мной
режим повторения

# Выключить
хватит повторять
выключить режим повторения
конец повторения
```

---

## Быстрые команды (fast_commands)

```
джарвис
эй джарвис
алло джарвис
джарвис есть работа
```

После активации можно произнести составную команду, например:

```
открой youtube
открой telegram
пауза youtube
```

---

## Завершение работы

```
джарвис пока
на сегодня всё
джарвис выключайся
```

---

# 🔙 Отмена команды

В любом режиме можно отменить действие и вернуться назад:

```
назад
отмена
отбой
передумал
забыли
```

---

# 📝 Файл заметок

Заметки хранятся в файле:

```
notes.txt
```

Каждая заметка записана на отдельной строке. Файл создаётся автоматически при добавлении первой заметки.

# 🛑 Завершение работы

Скажи команду завершения или введите `2` в меню консоли.

```
джарвис сегодня всё
на сегодня всё джарвис
```
